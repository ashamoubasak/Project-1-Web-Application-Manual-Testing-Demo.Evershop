## Project Overview
An e-commerce web application Demo Evershop is used in this project where users can browse products, purchase online and add to cart.
The purpose of this project is to test the usability and performance mainly the Search functionality manually. 
The full functionality of the project is manually tested using Postman.

---

## Project Objective
- Verify Verify that all major features of the web application work as expected.
- Identify and report bugs to improve application quality.
- Gain hands-on experience in manual testing processes and tools.


---

## Scope
- **In Scope:** Product Search, Cart Management, Checkout.
- **Out of Scope:** Backend server testing, API automation testing (not covered in this manual project).
